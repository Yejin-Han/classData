웹 표준(Web Standards)

웹 사용성(Web Usability) : 사용자가 웹 사이트를 사용하면서 얻는 사용자 경험(UX: User eXperience)과 웹 사이트를 편리하고 정확하게 사용할 수 있는가를 가리킨다.

웹 표준(Web Standards) : 월드 와이드 웹을 서술하고 정의하는 공식 표준이나 다른 기술 규격을 가리키는 일반적인 용어이다.

반응형 웹 디자인(Responsive Web Design) : 사용자의 환경과 행동 패턴에 유기적으로 적절하게 반응하는 웹 요소들에 대한 총체적 디자인을 말한다.

적응형 웹 디자인(Adaptive Web Design) : 몇 개의 해상도를 정의하여 CSS 코딩 시 브라우저가 범위 사이즈에 속하면 그에 따라 정의된 스타일로 표현한다.

월드 와이드 웹을 서술하고 정의하는 공식 표준이나 다른 기술 규격을 가리키는 일반적인 용어이다.
